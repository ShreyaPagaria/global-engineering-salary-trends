# global-engineering-salary-trends
Exploratory Data Analysis of global Data Science salaries (2020–2025) using the Kaggle Salary Dataset. Includes trends by seniority, role, company size, remote ratio, and geography with insightful visualizations
Kaggle Dataset used: https://www.kaggle.com/datasets/adilshamim8/salaries-for-data-science-jobs/data 

### Project Overview
This project analyzes the Kaggle dataset “Salaries for Data Science Jobs” to understand compensation patterns across the global data science workforce from 2020 to 2025. The notebook includes a structured Exploratory Data Analysis (EDA) approach with guiding questions, visualizations, and insights.

### Key Objectives
Examine salary distribution across countries and job roles
Compare compensation across experience levels (EN, MI, SE, EX)
Evaluate the impact of remote work (0%, 50%, 100%) on salary
Analyze trends over years (2020–2025)
Identify top-paying job titles and company-size patterns

### Tech Stack
Python (Numpy, Pandas), Matplotlib, seaborn
